# Clipboard History With CopyQ

Clipboard history setup for Ubuntu GNOME using CopyQ, as part of Linux Toolbox's setup-once workflow.

## Install

```bash
git clone https://github.com/HNam1234/linux-toolbox.git
cd linux-toolbox/clipboard-history
./install.sh
```

## Use

- Press `Super+V` to show clipboard history.
- Type to search.
- Select an item in CopyQ to reuse it.
- CopyQ starts automatically on login.

If CopyQ is not installed yet, the installer asks Ubuntu to install the `copyq` package.
